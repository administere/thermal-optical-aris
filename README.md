# 热光混合处理器 — 工程验证

> 独立工程分析。核心论点：热不是寄生效应，是计算机制本身。

## 这是什么

这个仓库是我对一种新型热光混合处理器架构的工程验证：

```
VCSEL 阵列 → DiSubPc·C70 热筛 (242°C) → CMOS+APD 探测器阵列
    ↑                ↑                        ↑
  Q 编码光子     光热 Δn 筛选             直接光电探测
                光子穿过热筛             (点积结果)
```

核心物理机制是**光子复用**：单个光子脉冲依次穿过 D 个调制点，完成 D 次乘累加（MAC）运算。光在这里做两件事——既承载 Q 编码信号，又把光热层加热到量子相干拍频窗口（242°C）。

## 六个工程问题

我围绕六个核心问题展开分析，逐一给出结论：

| # | 工程问题 | 结论 | 状态 |
|---|---------|------|:--:|
| 1 | VCSEL 光束能否自维持 242°C？ | 需约 16W 辅助加热，可进一步优化 | ⚠️ |
| 2 | 0.033Hz 权重更新是否可行？ | 静态推理场景可行；训练/多租户不行 | ✅ |

| 6 | 实验验证路径？ | $15K，4 周起；4×4 阵列是关键里程碑 | ⚠️ |

## 诚实评估

- **物理上自洽**：光子复用的 attojoule 优势是 Maxwell 方程的直接推论，不是外推。
- **架构新颖**：自由空间热筛在物理实现上不同于波导 MZI（西电 PTC）和被动衍射（Gezhi OGPU）。
- **不是通用处理器**：权重更新频率只有 ~0.033Hz，只适合静态推理。训练、LoRA、多租户切换都不支持。
- **Amdahl 定律约束**：自回归推理中，注意力只占层内约 3% 的浮点运算。全系统端到端加速约 1×。价值在注意力能耗的降低，不在端到端吞吐。
- **仿真到实验的鸿沟**：目前全是仿真，没有硬件。第一个实验：VCSEL + DiSubPc·C70 薄膜 + APD（约 $15K，4 周）。

## 原创性声明

本工作提出以下**首次公开**的技术贡献：

1. **首次将 DiSubPc·C70 有机共晶用于光学计算**。该材料由川大/中科院团队发现并发表于 *Nature Photonics* (2026)，但原论文只涉及光热转换（蒸汽、海水淡化、光热治疗），完全没有提到计算应用。我是第一个提出利用其 242°C / 17.6 GHz 量子相干拍频窗口做光学 MAC 运算的。

2. **首次提出自加热热筛架构**。承载数据的 VCSEL 光束同时把 DiSubPc·C70 薄膜维持在 242°C，不需要外部加热器。以往所有热光计算架构（PHIL、PCM-GEMM）都用外部电加热或独立加热光源。

3. **首次在自由空间热调制阵列中实现光子复用**。一个光子脉冲穿过 D 个调制点，完成 D 次乘累加。光子复用这个概念在波导延迟线里有先例（ReFOCUS, MICRO 2023），但在自由空间热筛中是我第一个提出来的。

详细的先例分析见 [PRIOR_ART.md](PRIOR_ART.md)。

## MZI 网格热-光耦合仿真包 (`mzi_mesh/`)

> COMSOL 等效替代：2D 有限差分热求解器 + 解析传输矩阵光学

构建了波导集成 MZI 网格的完整电-热-光耦合仿真框架，模拟 **电功率 → 焦耳热 → 温度分布 → Δn → 相移 → 光学传输矩阵** 的完整物理链。

### 物理模块

| 模块 | 文件 | 功能 |
|------|------|------|
| MZI 传输矩阵 | `mzi.py` | Clements 约定 T(θ,φ) = [[cosθ, -e^{-iφ}sinθ], [e^{iφ}sinθ, cosθ]] |
| 热光移相器 | `phase_shifter.py` | V → ΔT → Δn → Δφ，1D 分析热阻模型 |
| **2D 热求解器** | `thermal_2d.py` | ★ 有限差分稳态热方程，热串扰矩阵 C_{ij} |
| Clements 网格 | `clements_mesh.py` | N×N 酉矩阵分解/合成，N(N-1)/2 MZI |
| SVD 引擎 | `svd_engine.py` | 一般矩阵 M = U Σ V† |
| 保真度分析 | `fidelity.py` | 热串扰下矩阵保真度 ℱₐ、有效 bit 精度 |

### 关键数字（SOI 平台，1550 nm）

| 指标 | 值 |
|------|-----|
| Pπ（π 相移功耗） | **6.8 mW** |
| 75 μm 间距 → 热串扰 | α = 6.8%，有效 5.7 bit |
| 127 μm 间距 → 热串扰 | α = 1.6%，有效 7.7 bit |
| 200 μm 间距 → 热串扰 | α = 0.3%，有效 10.1 bit |
| 4×4 Clements 网格 @ 127 μm | ℱₐ = **0.9867** |
| N≤16 酉矩阵重建保真度 | ℱₐ = **1.000000** (无串扰) |
