## 这是什么

这个仓库是我对一种新型热光混合处理器架构的工程验证：

```
VCSEL 阵列 → DiSubPc·C70 热筛 (242°C) → CMOS+APD 探测器阵列
    ↑                ↑                        ↑
  Q 编码光子     光热 Δn 筛选             直接光电探测
                光子穿过热筛             (点积结果)
```

核心物理机制是**光子复用**：单个光子脉冲依次穿过 D 个调制点，完成 D 次乘累加（MAC）运算。光在这里做两件事——既承载 Q 编码信号，又把光热层加热到量子相干拍频窗口（242°C）。

### 关键发现

**1. 850nm 吸收: 低于带隙 (MOESM6 + MOESM1)**
光学带隙 E_g = 2.25 eV (551 nm)。VCSEL 波长 850 nm (1.46 eV) 远低于带隙，吸收依赖电荷转移带尾。
乌尔巴赫外推: α(850nm) ≈ **350 cm⁻¹** (不确定性 ±2×)，10μm 薄膜吸收率仅 ~**29%**，而非原假设的 60%。

**2. 调制不是单一的 (MOESM7 + MOESM8 + MOESM2)**
三种不同的调制机制，速度跨越 10 个数量级：

| 机制 | 时间尺度 | 最大频率 | 物理基础 | 数据来源 |
|------|---------|---------|---------|---------|
| 热 Δn | τ≈2s | ~0.08 Hz | 光热加热 → 晶格升温 → Δn | MOESM7 Fig 4a |
| 电子布居 | τ≈4.2ns | ~0.24 GHz | 激发态衰减 → Kramers-Kronig Δn | MOESM8 Fig 5d |
| 量子拍频 | T=56.8ps | **17.6 GHz** | S₁↔¹TT 相干振荡 | MOESM2 Reviewer Response |

**3. 非中心对称: C70 独有的优势 (MOESM5 CIF)**
2DiSubPc-C70 在极性 **Cc** 空间群中结晶 → 允许 χ⁽²⁾ 非线性。
因果链: Cc 极性空间群 → 非零 χ⁽²⁾ → 强相干态混合 → 17.6 GHz 拍频 → 242°C 光热转换。
C60 共晶是中心对称 P2₁/n → χ⁽²⁾ 禁止 → 拍频仅 6.0 GHz。

**4. 替代材料: DiSubPc·C70 仍是最优选择 (文献搜索)**
调查了 10 种候选材料 (有机 CT 共晶、克酮酸菁染料、GST/Sb₂S₃ PCM、MXene、MoS₂ TMD、卟啉 COF)。
**没有材料能同时满足** 242°C 工作温度 + 量子相干拍频 + 非中心对称 χ⁽²⁾ + 已验证的光热稳定性。

## 诚实评估

- **物理上自洽**：光子复用的 attojoule 优势是 Maxwell 方程的直接推论，不是外推。
- **架构新颖**：自由空间热筛在物理实现上不同于波导 MZI（西电 PTC）和被动衍射（Gezhi OGPU）。
- **存在真正的实验不确定性**：
  - 850nm 吸收系数从未被直接测量过（所有光谱在 800nm 处截止）
  - 调制带宽受限于激发态衰减（~0.24 GHz）还是量子拍频（17.6 GHz），未经验证
  - 242°C 时的量子相干失相时间未知
- **不是通用处理器**：权重更新频率 ~0.5Hz（热机制），仅适用于静态推理。训练、LoRA、多租户切换不支持。
- **Amdahl 定律约束**：自回归推理中，注意力仅占层内约 3% 的浮点运算。价值在注意力能耗降低，不在端到端吞吐。
- **仿真到实验的鸿沟**：目前全是仿真，没有硬件。第一优先级实验: 850nm 薄膜吸收率测量。

## 原创性声明

本工作提出以下**首次公开**的技术贡献：

1. **首次将 DiSubPc·C70 有机共晶用于光学计算**。该材料由川大/中科院团队发现并发表于 *Nature Photonics* (2026)，但原论文只涉及光热转换（蒸汽、海水淡化、光热治疗），完全没有提到计算应用。我是第一个提出利用其 242°C / 17.6 GHz 量子相干拍频窗口做光学 MAC 运算的。

2. **首次提出自加热热筛架构**。承载数据的 VCSEL 光束同时把 DiSubPc·C70 薄膜维持在 242°C，不需要外部加热器。以往所有热光计算架构（PHIL、PCM-GEMM）都用外部电加热或独立加热光源。

3. **首次在自由空间热调制阵列中实现光子复用**。一个光子脉冲穿过 D 个调制点，完成 D 次乘累加。光子复用这个概念在波导延迟线里有先例（ReFOCUS, MICRO 2023），但在自由空间热筛中是我第一个提出来的。

4. **首次识别非中心对称 Cc 空间群对量子相干拍频的结构基础** (v5 新增)。通过 MOESM3-5 CIF 文件的晶体学分析，揭示了 C70 共晶的极性空间群是 17.6 GHz 量子拍频的结构根源，而 C60 共晶（中心对称 P2₁/n）仅产生 6.0 GHz 拍频。
